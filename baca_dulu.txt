CD ini merupakan pelengkap dari buku 

"Membuat Website Profil Sekolah + PPDB Online"


- Final Project berada di folder FINAL PROJECT.

- Ujicoba program menggunakan Xampp 1.8.0 yang ada di folder INSTALLER.

- Bagi Anda yang ingin langsung menggunakan Website Profil Sekolah + PPDB Online, silahkan lihat langsung VIDEO TUTORIALnya.

- Apabila Video Tutorial tidak bisa dimainkan, silahkan install dulu K_Lite_Codec yang ada di folder INSTALLER.

- Website Profil Sekolah + PPDB Online bisa dijalankan melalui http://localhost/ppdb_online. 

- Sedangkan Administratornya melalui http://localhost/ppdb_online/user dengan Username: admin dan Password: admin.
